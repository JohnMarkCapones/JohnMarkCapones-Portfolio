# Public Assets Directory

This folder contains all static assets for the portfolio website.

## Required Files in `/public` root:

### Favicons & Icons
- **favicon.ico** - Main favicon (32x32 or 16x16 pixels)
- **icon.svg** - SVG icon for modern browsers
- **apple-touch-icon.png** - Apple touch icon (180x180 pixels)

### Social Media
- **og-image.png** - Open Graph image for social media previews (1200x630 pixels)
  - This appears when you share your portfolio on Twitter, LinkedIn, Facebook, etc.
  - Should contain your name, title, or a branded graphic

### Optional
- **site.webmanifest** - Web app manifest (can be auto-generated)
- **robots.txt** - SEO robots file (can be auto-generated)

## Project Images

Each project folder should contain screenshots/images of the project:
- `/projects/campus-connect/` - Campus Connect project images
- `/projects/xiraya/` - Xiraya project images
- `/projects/pharmacy/` - Pharmacy System project images
- `/projects/media-platform/` - Media Platform project images

See individual project folders for specific image requirements.
