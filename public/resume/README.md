# Resume Folder

Place your resume PDF file in this folder.

## Required File:
**John-Mark-Capones-Resume.pdf** - Your resume in PDF format

## File Naming:
- Use your full name in the filename
- Format: FirstName-LastName-Resume.pdf
- Example: John-Mark-Capones-Resume.pdf

## Tips:
- Keep the PDF file size reasonable (under 2MB)
- Ensure the PDF is readable and well-formatted
- Use a professional resume template
- Update regularly as you add new projects/skills

## How it works:
- The website will serve this PDF when users click "Download Resume" in the command palette (Ctrl/Cmd+K)
- Users can also download it from the system info page
