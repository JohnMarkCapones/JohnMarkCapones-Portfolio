# Pharmacy Management System Project Images

Place the following images in this folder:

1. **pos-system.png** - Screenshot of the Point of Sale (POS) system
2. **dashboard.png** - Screenshot of the main dashboard
3. **inventory.png** - Screenshot of the inventory management interface

## Recommended Image Specifications:
- Format: PNG or JPEG
- Size: 1920x1080 pixels (or similar 16:9 ratio)
- File size: Keep under 500KB each for optimal loading
- Quality: High resolution, clear screenshots

## Tips:
- Show the POS interface with sample transactions
- Capture inventory management features
- Ensure no real patient or medication data is visible
- Use dummy/test data for screenshots
