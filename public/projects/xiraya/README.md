# Xiraya E-Commerce Project Images

Place the following images in this folder:

1. **homepage.png** - Screenshot of the Xiraya homepage
2. **product-catalog.png** - Screenshot of the product catalog/listing page
3. **admin-dashboard.png** - Screenshot of the admin dashboard

## Recommended Image Specifications:
- Format: PNG or JPEG
- Size: 1920x1080 pixels (or similar 16:9 ratio)
- File size: Keep under 500KB each for optimal loading
- Quality: High resolution, clear screenshots

## Tips:
- Show the e-commerce functionality clearly
- Include product images and pricing if applicable
- Capture the modern UI design
- Ensure no sensitive customer data is visible
