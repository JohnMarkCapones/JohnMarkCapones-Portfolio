# Media Platform Project Images

Place the following images in this folder:

1. **architecture.png** - System architecture diagram or overview
2. **dashboard.png** - Screenshot of the media platform dashboard

## Recommended Image Specifications:
- Format: PNG or JPEG
- Size: 1920x1080 pixels (or similar 16:9 ratio)
- File size: Keep under 500KB each for optimal loading
- Quality: High resolution, clear screenshots

## Tips:
- For architecture.png: Create a clear system architecture diagram
- Show the media handling capabilities
- Capture the streaming or processing features
- Consider using diagrams for technical architecture
