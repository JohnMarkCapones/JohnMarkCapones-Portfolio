# Campus Connect Project Images

Place the following images in this folder:

1. **dashboard.png** - Screenshot of the Campus Connect dashboard
2. **quiz-system.png** - Screenshot of the quiz/assessment system
3. **student-management.png** - Screenshot of the student management interface
4. **mobile-app.png** - Screenshot of the mobile application

## Recommended Image Specifications:
- Format: PNG or JPEG
- Size: 1920x1080 pixels (or similar 16:9 ratio)
- File size: Keep under 500KB each for optimal loading
- Quality: High resolution, clear screenshots

## Tips:
- Take screenshots with clean, representative data
- Crop to show the most important features
- Consider using browser screenshots with a nice window frame
- Ensure no sensitive information is visible
